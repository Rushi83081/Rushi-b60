# VM Extensions

VM Extensions are small programs or agents that run inside a Virtual Machine to do tasks.

- They are used to perform specific tasks inside the VM.
- Common use cases:
  - Install software
  - Run scripts
  - Backup Virtual Machines

----

# Custom Data

**Custom Data** is a script or data that runs **only once** during the first VM boot.

- Used for initial setup.
- Runs automatically when the VM starts for the first time.

---

# Storage Account

A Storage Account in Azure provides storage services.

It is similar to storage services in AWS.

# AWS vs Azure Storage Services

| AWS Service  | Azure Service           |
|------------- |-------------------------|
| S3           | Blob Storage            |
| EFS          | Azure File              |
| SQS          | Azure Queue             |
| DynamoDB     | Azure Table Storage     |
