# VM Images and Sizes

## Image
- Image is similar to **AMI in AWS**.
- An image is a template that includes:
  - Operating System
  - Pre-installed software
  - Configuration settings

Images are used to launch Virtual Machines.

## VM Size (Instance Type)
VM size defines **CPU and RAM**.

### Common VM Series
- B Series
- D Series
- E Series
- F Series

Choose size based on application requirements.

---

# OS Disk Types

OS Disk is the storage used by a Virtual Machine.

### Disk Types

#### 1. Standard HDD
- Cost effective
- Slower performance
- Used for development and testing

#### 2. Standard SSD
- Balanced performance
- Used for web servers

#### 3. Premium SSD
- High performance
- Low latency
- Used for databases and production applications

---
# IOPS and Throughput

- **IOPS** means number of input/output operations per second.
- **Throughput** means data transfer speed.

Higher IOPS and throughput give better disk performance.
