# awk Command

## Description

* **awk** is used for pattern scanning and processing.
* It is mainly used for analyzing and reporting structured text like logs or CSV files.

## Syntax
```
awk '{ action }' file_name
```

## Example 1 (14.sh)
```
#!/bin/bash

awk '{ print $1, $2, $3 }' demo.log
```

## Example 2

- ### Print selected columns:
```
#!/bin/bash

awk '{ print $1, $3 }' logfile
```
