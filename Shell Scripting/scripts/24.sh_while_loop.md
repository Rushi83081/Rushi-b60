# while Loop
### Description

- A while loop executes code as long as the condition remains true.

## Example 1: Count Numbers
```
#!/bin/bash

count=1

while [ $count -le 5 ]
 do
   echo "Value is $count"
   count=$((count+1))
 done
```

**What this example does:**

- Prints numbers from 1 to 5 using a while loop.
--- 

## Example 2: Real-Time Monitoring
```
while true
 do
   df -h
   sleep 3
 done
```
**What this example does:**

- Continuously monitors disk usage every 3 seconds.
