# while Loop
### Description

- A while loop executes code as long as the condition remains true.

## Example 1: Count Numbers
```
#!/bin/bash

count=1
num=10

while [ $count -le $num ]
 do
   echo "Value is $count"
   let $count++
 done
```

**What this example does:**

- Prints numbers from 1 to 10 using a while loop.
