## if-elif-else Syntax
```
if [ condition ]
then
    # command
elif [ condition ]
then
    # command
elif [ condition ]
then
    # command
else
    # commands
fi
```

# Marks and Grade Script (if-elif-else)

## Example (20.sh)

```
#!/bin/bash

echo "Enter marks:"
read marks


if [ $marks -gt 80 ]
then
    echo "Grade A"
elif [ $marks -gt 60 ]
then
    echo "Grade B"
elif [ $marks -gt 40 ]
then
    echo "Grade C"
else
    echo "You are fail"
fi
```
```
bash 20.sh
```

- Assigns grades based on marks using if-elif-else conditional statements.
