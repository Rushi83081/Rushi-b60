## for Loop

**Description**

- A for loop runs a block of code for each value in a list or range.
---

## Example 1: Print Numbers
```
#!/bin/bash

for i in 1 2 3 4 5
 do
   echo "Number: $i"
 done
```
**What this example does:**

- Prints numbers from 1 to 5 one by one.
---

## Example 2: Print Numbers Using Range
```
#!/bin/bash

for i in {1..30}
 do
   echo "Number: $i"
 done
```
**What this example does:**

- Prints numbers from 1 to 30 using a range.
---

## Example 3: Loop Through Names
```
#!/bin/bash

for name in rushi rahul roshan ram
 do
   echo "Name: $name"
 done
```
**What this example does:**

- Prints each name from the given list.
