## 10. Read File Using User Input (10.sh)

### Example 1 

```
echo "Enter file name:"
read file

filecontents=$(cat "$file")  **Note:** Reads the entire content of the file from top to bottom.

echo "Contents:"
echo "$filecontents"
```

### Example 2

```
echo "Enter file name:"
read file

filecontents=$(head "$file") **Note:** Reads the first 10 lines of the file

echo "Contents:"
echo "$filecontents"
```

### Example 3 

```
echo "Enter file name:"
read file

filecontents=$(tail -3 "$file") **Note:** Reads the last 3 lines of the file.

echo "Contents:"
echo "$filecontents"
```
