## Checking Service Status Using Function
```
#!/bin/bash

check_service()
{
    service=$1
    if systemctl is-active $service 
    then
        echo "$service is running"
    else
        echo "$service is NOT running"
    fi
}

check_service nginx
check_service ssh
check_service httpd
```

- **Explanation:** Checks whether a service is running using systemctl.
