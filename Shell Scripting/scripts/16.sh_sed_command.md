# sed Command

## Description

* **sed (Stream Editor)** is used to edit text in a stream or file.
*  It can replace, delete, or modify lines.

## Syntax
```
sed 'command' file_name
```

## Example 1 (16.sh)

- ### Replace text:

```
#!/bin/bash

sed 's/WARNING/ERROR/g' demo.txt
```

## Example 2

- ### On a log file:

```
#!/bin/bash

sed 's/WARNING/ERROR/g' logfile
```

```
bash 16.sh
```
