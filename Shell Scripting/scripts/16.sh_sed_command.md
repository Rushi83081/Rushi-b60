# sed Command

## Description

* **sed (Stream Editor)** is used to edit text in a stream or file.
*  It can replace, delete, or modify lines.

## Syntax
```
sed 'command' file_name
```

## Example 1 (16.sh)

- ### Replace text:

```
#!/bin/bash

sed 's/WARNING/ERROR/g' demo.txt
```
```
bash 16.sh
```

## Example 2

- ### On a log file:

```
#!/bin/bash

sed 's/WARNING/ERROR/g' logfile
```

```
bash 16.sh
```

# Combining grep, awk, and sed

## Example Script
```
#!/bin/bash

grep "WARNING" logfile | awk '{ print $1, $2, $3 }' | sed 's/WARNING/ERROR/g'
```
```
bash 16.sh
```

### This will:

  - **Search for WARNING**

  - **Print selected columns**

  - **Replace WARNING with ERROR**
