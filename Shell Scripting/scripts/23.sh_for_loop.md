## 1. Read Names from a File
* **Create File**
 ``` 
cat > names.txt
```
```
rushi
rahul
roshan
ram
sham
```
## Script
```
#!/bin/bash

file="names.txt"

for name in $(cat $file)
 do
   echo "Name is $name"
 done
```
### What this example does:

- Reads names from a file and prints them one by one.
---

## 2. for Loop with Command Line Arguments

### Example 2

```
#!/bin/bash

for user in $1 $2 $3 $4 $5
 do
   useradd $user
   echo "User $user is created"
 done
```
**Run command:**
```
bash 23.sh rushi ram sham rahul rohan
```

## What this example does:

- Creates multiple users using command-line arguments.
---

## 3. Extra: Backup Files Using for Loop (Real-time)
### Example

```
#!/bin/bash

for file in /var/log/*.log
 do
   cp $file /backup/
   echo "Backed up: $file"
 done
```

**What this example does:**

- Takes backup of all .log files from /var/log directory.
