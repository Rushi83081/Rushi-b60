**Argument passing** means sending values to a shell script from the command line while executing it.

###ðŸ”¹ Syntax
```

```

### ðŸ”¹ Special Variables

| Variable | Meaning                   |
| -------- | ------------------------- |
| `$0`     | Script name               |
| `$1`     | First argument            |
| `$2`     | Second argument           |
| `$3`     | Third argument            |
| `$@`     | All arguments             |
| `$#`     | Total number of arguments |

## ðŸ”¹ Example (7.sh)
```

```
