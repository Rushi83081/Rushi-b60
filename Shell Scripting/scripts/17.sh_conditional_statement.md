# Control Structures in Shell Script

## Types

  1. **Conditional Statements (if, if-else, elif)**

  2. **Loops (for, while, until)**


# Conditional Statements

## Description

- **Conditional statements** are used for decision making.

### if-else Syntax

```
if [ condition ]
then
    commands
else
    commands
fi
```

## Example (17.sh)
```
#!/bin/bash

echo "Enter your age:"
read age


if [ $age -ge 18 ]
then
    echo "You can vote"
else
    echo "You are not eligible"
fi
```
```
bash 17.sh
```
