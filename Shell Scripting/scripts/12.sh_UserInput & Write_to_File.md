* ## Task

    - Take file name as input from the user.

    - Take content from the user.

    - Store the content inside the given file.

### Answer 

```
echo "Enter file name:"
read file

echo "Enter content (Press Ctrl + D to finish):"
cat >> "$file"

echo "success"
```


* read takes input from the user.

* **Ctrl + D** ends the input.

* **cat >> file** appends user content to the file.
