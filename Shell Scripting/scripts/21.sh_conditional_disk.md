# Disk Usage Monitoring Script

## Example (21.sh)
```
#!/bin/bash

disk=85

if [ $disk -ge 90 ]
then
    echo "CRITICAL: Disk usage very high"
elif [ $disk -ge 75 ]
then
    echo "WARNING: Disk usage high"
else
    echo "Normal"
fi
```
```
bash 21.sh
```

- Monitors disk usage percentage and displays NORMAL, WARNING, or CRITICAL status.
